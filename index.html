Perguntas e Respostas: Teoria da Computação

Parte 1: Automatos e Linguagens Regulares

1. Qual dos seguintes modelos é mais apropriado para descrever linguagens regulares?

  • a) Gramáticas livres de contexto
  • b) Autómatos a pilha
  • c) Máquinas de Turing
  • d) Autómatos finitos

2. Qual é a operação que mantém uma linguagem regular regular?

  • a) União com uma linguagem livre de contexto
  • b) Interseção com uma linguagem regular
  • c) Concatenar com uma linguagem não regular
  • d) Interseção com uma linguagem recursiva

3. Para provar que uma linguagem não é regular, utiliza-se frequentemente:

  • a) O lema do bombeamento
  • b) A equivalência entre autómatos finitos determinísticos e não determinísticos
  • c) A forma normal de Chomsky
  • d) O problema da parada

4. Se um autómato finito não determinístico (AFN) possui \(n\) estados, o autómato finito determinístico (AFD) equivalente pode ter até:

  • a) \(n+1\) estados
  • b) \(2^n\) estados
  • c) \(n^2\) estados
  • d) \(n!\) estados

Parte 2: Linguagens Livres de Contexto

5. Qual dos seguintes problemas é decidível para gramáticas livres de contexto?

  • a) Determinar se uma linguagem é vazia
  • b) Verificar se duas gramáticas geram a mesma linguagem
  • c) Determinar se uma gramática gera uma linguagem regular
  • d) Determinar se uma palavra faz parte de uma linguagem irrestrita

6. Qual a característica que define um autómato a pilha em comparação aos autómatos finitos?

  • a) Utiliza duas fitas de memória
  • b) Reconhece linguagens recursivamente enumeráveis
  • c) Possui uma pilha para memória adicional
  • d) Funciona apenas de forma determinística

7. A forma normal de Chomsky para gramáticas livres de contexto exige que:

  • a) Todas as regras de produção contenham pelo menos dois terminais
  • b) Cada regra de produção seja na forma \( A \rightarrow BC \) ou \( A \rightarrow a \)
  • c) Todas as produções incluam símbolos não terminais
  • d) A gramática seja completamente não ambígua

8. O lema do bombeamento para linguagens livres de contexto indica que, para uma linguagem livre de contexto \(L\), qualquer palavra \(w \in L\) longa o suficiente pode ser dividida em:

  • a) \(w = uvx\), onde \(uv\) pode ser bombeado
  • b) \(w = uvxy\), onde \(v\) e \(y\) podem ser bombeados
  • c) \(w = uxyz\), onde \(u\) pode ser bombeado
  • d) \(w = vx\), onde \(v\) pode ser bombeado

Parte 3: Teoria da Computabilidade

9. Qual é a função principal da Máquina de Turing no contexto da teoria da computação?

  • a) Determinar a diferença entre linguagens regulares e livres de contexto
  • b) Fornecer um modelo abstrato para descrever a computação
  • c) Resolver problemas decidíveis em tempo exponencial
  • d) Garantir que todas as linguagens regulares sejam reconhecíveis

10. Qual das opções abaixo descreve uma característica de uma linguagem recursiva?

  • a) Existe uma máquina de Turing que a reconhece, mas pode não parar em entradas fora da linguagem.
  • b) É equivalente a uma linguagem regular em termos de poder computacional.
  • c) Existe uma máquina de Turing que a reconhece e sempre para em qualquer entrada.
  • d) Apenas linguagens livres de contexto podem ser recursivas.

  • b) A sequência de símbolos terminais gerados

13. O problema da parada é indecidível porque:

  • a) Ele requer mais memória do que está disponível em uma Máquina de Turing.
  • b) Não é possível prever se um programa entrará num ciclo infinito para todas as entradas.
  • c) Algumas linguagens não podem ser representadas por uma Máquina de Turing.
  • d) Todas as opções acima.

14. Qual dos seguintes problemas é decidível para autómatos finitos?

  • a) Determinar se o autómato aceita uma linguagem infinita.
  • b) Decidir se dois autómatos aceitam linguagens idênticas.
  • c) Determinar se o autómato rejeita todas as entradas.
  • d) Verificar se um autómato finito determinístico é equivalente a uma Máquina de Turing.

  • a) Ele prova que linguagens regulares possuem um número infinito de palavras.
  • b) Ele fornece uma forma de construir novos autómatos.
  • c) Ele garante que strings longas em uma linguagem regular podem ser repetidas.
  • d) Ele identifica um limite superior para o tamanho de linguagens regulares.
